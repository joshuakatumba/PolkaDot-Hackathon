# 🏦 SmartSave (AutoTreasury)

[![Polkadot Ecosystem](https://img.shields.io/badge/Polkadot-Ecosystem-E6007A?logo=polkadot)](https://polkadot.network/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> **"Save once. Spend your yield. Your principal is protected."**

SmartSave is a decentralized hybrid savings and cross-chain payment platform designed for the Polkadot ecosystem. It enables users to protect their principal assets while putting their accrued yield to work across multiple parachains via XCM (Cross-Chain Message passing).

---

## 🚀 Overview

In the traditional world, savings are often "locked" or "liquid." SmartSave creates a third path: **Productive Liquid Savings**. 

By depositing DOT into the SmartSave vault, your principal is secured and automatically routed to optimized yield strategies. Instead of just letting that yield sit, SmartSave allows you to spend it directly for real-world or on-chain services—like paying for utility bills, subscriptions, or airtime—without ever touching your initial savings.

---

## ✨ Key Features

- **🛡️ Principal Protection**: Your "Saved" amount is never spent. Only the yield generated by your assets is used for payments.
- **🌐 Native XCM Integration**: Powered by Polkadot's Cross-Chain Message passing, the vault can swap assets on Asset Hub and send payments to any parachain.
- **📈 Automated Yield Strategies**:
  - **Native Staking**: Direct interaction with the Polkadot Relay chain staking precompiles.
  - **DeFi Lending**: Automated lending on Asset Hub to capture the best risk-adjusted returns.
- **📊 Real-time Monitoring**: A robust backend tracks every deposit, harvest, and cross-chain payment.
- **💎 Premium UX**: A sleek, Next.js-powered frontend with "Glassmorphism" design.

## 🚦 Getting Started

### Prerequisites
- Node.js v18+
- Hardhat
- A Polkadot-compatible wallet (e.g., Talisman, SubWallet)

### Quick Start

1. **Clone & Install**
   ```bash
   git clone [your-repo-link]
   cd AutoTreasury
   ```

2. **Smart Contracts**
   ```bash
   cd contracts
   npm install
   npx hardhat compile
   ```

3. **Backend API**
   ```bash
   cd backend
   npm install
   cp .env.template .env # Update with your keys
   npm run dev
   ```

4. **Frontend Dashboard**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
